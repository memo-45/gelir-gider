⚡ Supabase Kurulumu

Verilerin bulutta saklanması ve her cihazdan erişilebilmesi için Supabase bilgilerini gir. Tamamen ücretsiz, 5 dakikada kurulum.

  1. supabase.com → New Project oluştur (ücretsiz)
  2. Sol menü → SQL Editor → aşağıdaki kodu yapıştır ve çalıştır:
    create table if not exists islemler (id text primary key, tarih text, tutar numeric, tur text, aciklama text, kategori text); create table if not exists sabitler (id text primary key, gun int, tutar numeric, tur text, aciklama text, kategori text, "not" text); create table if not exists taksitler (id text primary key, tutar numeric, sayi int, tur text, aciklama text, kart text, baslangic text, odenmis_mi jsonb); alter table islemler enable row level security; alter table sabitler enable row level security; alter table taksitler enable row level security; create policy "anon_all" on islemler for all using (true) with check (true); create policy "anon_all" on sabitler for all using (true) with check (true); create policy "anon_all" on taksitler for all using (true) with check (true);
  3. Project Settings → API → Project URL ve anon public key kopyala
